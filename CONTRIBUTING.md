# Contributing

Спасибо за интерес к проекту.

## Как предложить изменение

1. Создай fork репозитория.
2. Создай отдельную ветку под изменение.
3. Установи зависимости и запусти тесты.
4. Отправь pull request с коротким описанием:
   - что изменено;
   - почему это нужно;
   - как проверялось.

## Локальная проверка

```bash
python -m pytest -v
```

Для изменений интерфейса полезно дополнительно запускать ручные проверки:

```bash
python tests/ui/test_dial_visual.py
python tests/ui/test_window_manual.py
python tests/services/test_tray_manual.py
```

## Стиль изменений

- сохраняй простую структуру проекта;
- не смешивай рефакторинг и функциональные изменения без необходимости;
- обновляй документацию, если меняется поведение приложения.
